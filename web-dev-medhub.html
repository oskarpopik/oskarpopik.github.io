medhub

GitHub repository link: https://github.com/oskarpopik/wdf-project-2024

Skills: Node.js and Express.js (back-end), SQLite3 (Database management), bcrypt (authentication), JavaScript, HTML and CSS (front-end), GitHub (version control)

Tools used: VSCode, GitHub, Bootstrap, npm

MedHub is a platform designed for managing medical professionals' profiles and treatment records. It simplifies the organization and retrieval of key data for both healthcare providers and administrators. Doctors can log in to manage their patients, while administrators oversee the system.

The MedHub platform is powered by Node.js and Express, providing robust server-side logic and efficient routing for handling user requests. For data storage and management, SQLite3 is used, allowing for structured storage of medical professional profiles, patient records, and treatment data. On the front-end, Handlebars is employed to create dynamic views, enabling real-time updates and seamless navigation across different sections of the platform. To ensure user security, bcrypt is integrated for password hashing, safeguarding sensitive login credentials and preventing unauthorized access. The platform's design is made visually appealing through extensive CSS stylization, utilizing the Bootstrap framework to ensure a responsive and modern user interface.

MedHub offers an efficient and privacy-focused solution for medical administration.
