trivin

A group project done in collaboration with: Julia Can, Tasa Kriznar and Paula Lucane.

Visit the website: https://ju-nmd2023.github.io/wuid-project-group-12/

Interactive prototype and the design system: link to Figma

Skills: UX/UI design, JavaScript, HTML and CSS (front-end), GitHub (version control)

Tools used: VSCode, GitHub, Adobe Illustrator, Adobe Photoshop, Figma

TriVin is a mockup website that was created for a premium wine reseller.

The project involved multiple design stages, starting with low-fidelity wireframes, followed by a basic Figma prototype, and finally developing a fully interactive design.

After completing the prototype, the website was built using HTML and CSS, ensuring full responsiveness across all screen sizes. The site also incorporates simple CSS animations, with JavaScript used to enhance certain interactive features.
