change of heart

A group project done in collaboration with Paula Lucane.

Play the game: https://ju-nmd2023.github.io/fop-final-project-group-02/

Watch the game trailer: external link on Microsoft Stream (SharePoint)

Skills: JavaScript, HTML and CSS, GitHub (version control)

Tools used: VSCode, GitHub, Adobe Illustrator, Adobe Premiere Pro

This is top-down action-adventure game with some puzzle elements. The game is composed of 3 levels in unique locations and some quiz elements between them.

Instructions: Use arrow keys to control the character. Avoid foes and answer the questions correctly to progress through the game.
